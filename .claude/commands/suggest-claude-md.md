# CLAUDE.md 改善提案コマンド

このセッションの会話履歴を分析して、プロジェクトのCLAUDE.mdファイルに追加すべき内容を提案してください。

## 分析観点

以下の観点で会話履歴を分析してください：

### 1. 頻繁に使用されるパターン
- よく実行されるコマンドやワークフロー
- 繰り返し参照されるファイルやディレクトリ
- 定型的な作業手順

### 2. プロジェクト固有のルール
- コーディング規約や命名規則
- ファイル配置のルール
- コミットメッセージの形式

### 3. 学習した教訓
- エラーハンドリングで発見したこと
- パフォーマンス改善のヒント
- セキュリティ上の注意点

### 4. 効率化の機会
- 自動化できる作業
- カスタムコマンドにすべき操作
- hooksで対応できるチェック

## 出力形式

以下の形式で提案を出力してください：

```markdown
## 提案: [カテゴリ名]

### 追加推奨内容
[CLAUDE.mdに追加すべき具体的な内容]

### 理由
[なぜこの内容が有用か]

### 実装例
[具体的なコードや設定の例]
```

## 注意事項

- 機密情報（APIキー、パスワード等）は含めないでください
- 一般的すぎる内容より、このプロジェクト固有の内容を優先してください
- 実用的で即座に適用できる提案を心がけてください
