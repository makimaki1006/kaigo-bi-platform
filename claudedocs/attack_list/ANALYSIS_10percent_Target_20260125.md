**目的**: 標準シナリオ7.0%から目標10.0%へのギャップ+3.0ppを埋める施策探索

## 2. 10%以上を実現しているセグメント

### 2.1 実績データから確認済みの高成約率セグメント

| セグメント | 成約率 | N数 | 利用可能性 |
|-----------|--------|-----|-----------|
| **決裁者到達時** | **15.5%** | 58 | ◎ 最大レバー |
| 株式会社×訪問介護（九州） | 66.7% | 3 | △ N極少で不安定 |
| 有限会社（九州） | 9.1% | 11 | ○ 2,788件で限定的 |
| 有限会社（全国） | 12.7% | 119 | 参考値 |
| 医療クリニック(11-30人)（全国） | 9.4% | - | ○ |
| 宮崎県（九州） | 9.3% | 75 | ○ 地域限定 |
| 障害福祉（全国歴史的） | 93.3% | 15 | × 九州にほぼなし |

### 2.2 核心的発見

**決裁者到達率が最大のレバー**

| 到達状況 | 成約率 | 差 |
|---------|--------|-----|
| 決裁者到達 | 15.5% | - |
| 決裁者未到達 | 3.6% | - |
| **差分** | **+11.9pp** | 到達するだけで3倍以上 |

現状の決裁者到達率はわずか**5.7%**。これを改善することが10%到達の鍵。

## 3. 10%到達シナリオ分析

### 3.1 シナリオA: 決裁者到達率の劇的改善

計算式: 追加効果 = 到達率増分 × (到達時成約率 - 未到達時成約率)

| 到達率 | 増分 | 効果計算 | 期待成約率 |
|--------|------|---------|-----------|
| 5.7%（現状） | - | ベースライン | 7.0% |
| 10% | +4.3pp | +4.3% × 11.9% = +0.5pp | 7.5% |
| 15% | +9.3pp | +9.3% × 11.9% = +1.1pp | 8.1% |
| 20% | +14.3pp | +14.3% × 11.9% = +1.7pp | 8.7% |
| 25% | +19.3pp | +19.3% × 11.9% = +2.3pp | 9.3% |
| **30%** | +24.3pp | +24.3% × 11.9% = **+2.9pp** | **9.9% ≈ 10%** |

**結論**: 決裁者到達率を30%にすれば10%到達可能（現状の5.3倍）

### 3.2 シナリオB: A-listの極限絞込み

| 絞込条件 | 件数 | 期待成約率 | 課題 |
|---------|------|-----------|------|
| A-list全体 | 6,000件 | 7-10% | - |
| A-list × 宮崎県のみ | ~800件 | 9-11% | 件数不足 |
| A-list × 有限会社のみ | ~500件 | 10-13% | 件数不足 |
| A-list × 決裁者情報あり | ~350件* | **12-15%** | 件数極少 |

*要Salesforce調査

**課題**: 月間架電5,000-8,000件に対して件数が極端に不足

### 3.3 シナリオC: リード流入経路の改善（根本対策）

| 施策 | 期待効果 | 実現難易度 | コスト |
|------|---------|-----------|--------|
| 代表者直通リストの購入/獲得 | 到達率→50%+ | 中 | 高 |
| インバウンドリード活用 | 成約率15-20% | 中 | 中 |
| 紹介・リファラル強化 | 成約率20-30% | 高 | 低 |
| ウェビナー/セミナー集客 | 成約率15-25% | 中 | 中 |
| 展示会リード | 成約率10-15% | 中 | 高 |

## 4. 具体的な10%到達施策パッケージ

### 4.1 施策構成

```
┌─────────────────────────────────────────────────────────┐
│ 基盤施策（現計画）                          → +2.7pp   │
│ ・通所系除外、A/B/Cリスト運用                          │
│ ・期待成約率: 7.0%                                     │
├─────────────────────────────────────────────────────────┤
│ 追加施策①: 代表者直通リスト最優先                      │
│ ・PresidentName__c 入力済みリードを最優先消化          │
│ ・ミイダス等の代表者情報付きリードを優先               │
│ ・効果: +1.0〜1.5pp                                    │
├─────────────────────────────────────────────────────────┤
│ 追加施策②: インバウンド枠の確保                        │
│ ・月間商談100件中20件をインバウンドに割当              │
│ ・インバウンド成約率15%、アウトバウンド6%想定          │
│ ・加重平均: (80×6% + 20×15%) / 100 = 7.8%             │
│ ・効果: +0.8pp                                         │
├─────────────────────────────────────────────────────────┤
│ 追加施策③: 決裁者アポ取りスクリプト改善                │
│ ・「担当者様お願いします」→「代表の○○様いらっしゃいますか」│
│ ・代表者名を事前調査して架電                           │
│ ・到達率5.7%→15%で効果+1.1pp                          │
├─────────────────────────────────────────────────────────┤
│ 【合計】 7.0 + 1.2 + 0.8 + 1.1 = 10.1%                │
└─────────────────────────────────────────────────────────┘
```

### 4.2 施策優先度マトリクス

| 優先度 | 施策 | 期待効果 | コスト | 実行容易性 |
|--------|------|---------|--------|-----------|
| **1** | 決裁者名判明リードの最優先架電 | +1.0pp | 低 | 高 |
| **2** | インバウンド枠20%確保 | +0.8pp | 中 | 中 |
| **3** | 決裁者到達トークスクリプト改善 | +1.1pp | 低 | 中 |
| 4 | 代表者直通リスト購入 | +1.5pp | 高 | 高 |

**施策1-3を全て実行すれば 7.0% + 2.9pp = 9.9% ≈ 10%**

## 5. Salesforce調査項目

### 5.1 代表者情報付きリードの件数

```sql
-- 調査クエリ1: 代表者名入力済みリード
SELECT COUNT(Id) FROM Lead
WHERE PresidentName__c != null
AND IsConverted = false
AND Phone != null

-- 調査クエリ2: 株式会社×代表者名あり×訪問系
SELECT COUNT(Id) FROM Lead
WHERE LegalPersonality__c = '株式会社'
AND PresidentName__c != null
AND ServiceType1__c LIKE '%訪問%'
AND IsConverted = false

-- 調査クエリ3: 決裁者商談（Hearing_Authority__c）の内訳
SELECT Hearing_Authority__c, COUNT(Id), SUM(CASE WHEN IsWon THEN 1 ELSE 0 END)
FROM Opportunity
WHERE IsClosed = true
AND CloseDate >= 2025-04-01
GROUP BY Hearing_Authority__c
```

### 5.2 追加調査候補
