## 1. セグメント定義

### 1.1 優先度マトリクス

| 優先度 | 法人格 | サービス種別 | 理由 |
|-------|--------|-------------|------|
| **S** | 株式会社 | 訪問系 | タイプC成約率35.7%（最高） |
| **A** | 株式会社 | その他 | 株式会社はタイプC成約率21% |
| **B** | その他法人 | 訪問系/通所系 | サービス種別で救える |
| **C** | その他法人 | 入所系/その他 | ベースライン |

### 1.2 従業員規模セグメント

| 規模 | 定義 | 特徴 |
|-----|------|------|
| **小規模** | ~50人 | 発生率高いが成約率やや低い |
| **中規模** | 51-200人 | 発生率・成約率ともにバランス良（スイートスポット） |
| **大規模** | 200人+ | 成約率高いが発生率低い |
| **規模不明** | データなし | 全体の59%を占める |

### 1.3 架電可能条件（除外フィルタ）

以下に該当するアカウントは**全て除外済み**：

| 条件 | フィールド | 値 |
|-----|-----------|---|
| 商談中 | Status__c | '%商談中%' を含む |
| プロジェクト進行中 | Status__c | '%プロジェクト進行中%' を含む |
| 深耕対象 | Status__c | '%深耕対象%' を含む |
| 過去客 | Status__c | '%過去客%' を含む |
| グループ案件進行中 | RelatedAccountFlg__c | 'グループ案件進行中' |
| グループ過去案件実績あり | RelatedAccountFlg__c | 'グループ過去案件実績あり' |
| アプローチ禁止 | ApproachNG__c | true |
| 架電対象外 | CallNotApplicable__c | true |
| 電話番号なし | Phone | null |

## 8. データ取得用SOQLクエリ

### 8.1 セグメント別成約率取得

```sql
-- 優先度S × 中規模の成約率
SELECT
    COUNT(Id) total,
    SUM(CASE WHEN IsWon = true THEN 1 ELSE 0 END) won
FROM Opportunity
WHERE Account.LegalPersonality__c = '株式会社'
  AND Account.ServiceType__c LIKE '%訪問%'
  AND Account.NumberOfEmployees > 50
  AND Account.NumberOfEmployees <= 200
  AND CreatedDate >= 2026-01-01T00:00:00Z
  AND IsClosed = true
```

### 8.2 タイプC発生率取得

```sql
SELECT
    COUNT(Id) total,
    SUM(CASE WHEN Hearing_Authority__c = 'あり' THEN 1 ELSE 0 END) type_c
FROM Opportunity
WHERE Account.LegalPersonality__c = '株式会社'
  AND Account.ServiceType__c LIKE '%訪問%'
  AND Account.NumberOfEmployees > 50
  AND Account.NumberOfEmployees <= 200
  AND CreatedDate >= 2026-01-01T00:00:00Z
```
