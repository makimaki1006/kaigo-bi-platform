## 3. リスト構成（ローラー作戦用）

### 3.1 A/B/Cリスト分割

| リスト | 構成条件 | 件数 | 期待成約率 |
|--------|---------|------|-----------|
| **A-list** | 株式会社×訪問系(50万未満) + 有限会社全域 + 宮崎県×株式/社福/有限 + 社福×訪問系 | **~6,000件** | **7-10%** |
| **B-list** | 株式会社(50万未満・非訪問) + 社福全域 + 医療クリニック(11-30人) + 大分県×全法人格 | ~14,000件 | **5-7%** |
| **C-list** | 法人格null全域 + 残り医療法人(31-200人) + 他県×その他法人格 | ~10,000件 | **3-5%** |
| **除外** | 通所系 + 短期入所 + 医療大病院(201人+) + 人口50万以上 + 小規模多機能 | ~6,000件 | - |

**⚠️ A-list拡大理由（逆証明#41）**:
- 旧方針: 3,000件 → 月間架電5,000-8,000件で2週間未満で枯渇
- 新方針: 6,000件 → 約1ヶ月分を確保

**⚠️ 法人格null（~9,300件）はC-listに配置し、除外しない**:
- 受注の72%がここから発生する現実を尊重
- ただしA/B-listを先に消化することで実質的に後回し

### 3.2 架電順序

**A-list → B-list → C-list の順にローラー**

### 3.3 リサイクルルール

| 条件 | リサイクル間隔 |
|------|--------------|
| 不在（電話に出ない） | 2週間後に再架電 |
| 担当者不在 | 1週間後に再架電 |
| 検討中/資料送付済 | 3日後にフォロー |
| 明確な拒否 | 3ヶ月後（C-listに移動） |
| 全リスト一巡 | 3ヶ月後にA-listから再開 |

### 4.4 達成シナリオ

| シナリオ | 条件 | 期待成約率 | 信頼度 |
|---------|------|-----------|--------|

### 4.5 効果積算の留意事項

- 各施策効果は**独立ではなく重複あり**（例: 通所除外後のA-list優先効果は限定的）
- 決裁者到達率10%は**株式会社に限れば現状実績10.3%で既に達成済み**。社福は8%が限界
- 季節性あり: 7月(8.1%)、10月(8.7%)にスパイク。4-6月は2-3%で低迷する傾向
- **リスト縮小→商談化件数減少の相殺リスク**: ローラーなので架電数は不変。リスト一巡が早まるだけ（リサイクルで対応）

## 7. リスク要因

| リスク | 影響 | 対策 |
|--------|------|------|
| リスト枯渇（除外後~30,000件） | 4-6ヶ月で一巡 | リサイクル(3ヶ月)+新規リード流入 |
| N数不足による判断誤り | 小セグメントの過信 | 四半期で実績検証（N≧300） |
| 決裁者到達率が上がらない | 効果⑤が発現しない | 保守的シナリオ(6.0%)で計画 |
| 九州/全国比率(0.63)の変動 | 期待値とのズレ | 四半期で補正係数見直し |
| 季節変動（4-6月低迷） | 月次目標未達 | 四半期累計で評価、月単位で一喜一憂しない |
| 法人格null付与が進まない | C-listの再分類不能 | 現状のまま維持（除外しない） |
| リスト縮小→商談化減少 | 成約率UP×件数DOWNの相殺 | 架電数一定を維持、リサイクルで補填 |
