## 2. 因果（推定、根拠の強さ付き）

### 2-1. 【強い因果】プロダクト強化 → 再商談×代表者の増加

| 根拠 | 数字 |
|------|------|
| 再商談×代表者 | 1月0件 → 2月11件 |
| 再商談の中心チーム | 深堀チーム7件（再商談の58%） |
| 受注増+18件への寄与 | +11件（61%） |

**なぜ「強い因果」と言えるか:**
- 再商談は過去に断られた先への再アプローチ。「プロダクトが変わった」以外に代表者が再度話を聞く理由が乏しい
- 1月は再商談×代表者が0件 → 2月に突然11件。プロダクトアップデートのタイミングと整合する
- 深堀チームに集中（7/11件）→ 特定チームがプロダクト変更を活用した再アプローチ戦術を実行

### 2-2. 【中程度の因果】ターゲティング改善 → 「マッチしない」失注の減少

| 根拠 | 数字 |
|------|------|
| 「採用戦略とマッチしない」失注 | 1月21件 → 2月8件（-13件） |
| 「導入ニーズがない」失注 | 1月53件 → 2月44件（-9件） |

**なぜ「中程度」か:**
- ターゲティングが改善した（リストの質が上がった）とも解釈できる
- 単に商談数が減った（1月602件→2月610件でほぼ同数なので、この説明は弱い）
- リスト施策が直接関係しているかは、リード→商談のリンクがないため確定不可

### 2-3. 【中程度の因果】リストの質 → 代表者到達率の向上

| 根拠 | 数字 |
|------|------|
| 代表者商談の受注 | 1月8件 → 2月20件（+12件） |
| うち再商談×代表者 | 0件 → 11件 |
| うち新規×代表者 | 8件 → 7件（横ばい） |

**なぜ「中程度」か:**
- 代表者商談の増加+12件のうち、+11件は再商談。**新規の代表者アクセスは増えていない**
- リストの質（代表者直通番号）が効いたのは「新規開拓」ではなく「**過去に代表者と接点がある先の再アプローチ**」
- つまりリストの質単体ではなく、「リスト×プロダクト強化」の掛け合わせ

### 2-4. 【弱い因果】架電量の増加 → 受注増

| 根拠 | 数字 |
|------|------|
| 架電済リード | 1月3,633 → 2月5,328（+47%） |
| 受注 | 1月27件 → 2月45件（+67%） |

**なぜ「弱い」か:**
- リードと商談のデータリンクが不在（ConvertedOpportunityId未設定）
- 架電がどのチームの商談に繋がったか追跡不可
- 12月は架電4,335件で受注52件と2月より効率が良い → 架電量だけでは説明不足
