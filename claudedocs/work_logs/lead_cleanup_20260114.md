## 3. 人口データ補完

### 3.1 問題
人口データなしLead: 12,180件

原因分析:
- 住所(Street)が空: 5,760件
- 住所ありだがマッチ失敗: 6,420件

### 3.2 原因
政令指定都市の区名形式の不一致:
- Salesforce: `堺市東区`、`福岡市中央区`
- マスタ: `堺市`、`福岡市`（市名のみ）

### 3.3 対処
フォールバックロジックを実装:
```python
# 「堺市東区」→「堺市」にフォールバック
if city_name in DESIGNATED_CITIES:
    key = final_pref + city_name  # 市名のみでマッチ
```
