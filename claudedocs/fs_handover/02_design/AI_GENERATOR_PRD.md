# FS引継ぎメモ AI自動生成システム - 要件定義書（PRD）

## 1. エグゼクティブサマリー

### 1.1 プロジェクト概要
商談の文字起こしや先方との会話記録から、AIがFS引継ぎメモを自動生成するシステム

### 1.2 解決したい課題

| 現状の課題 | 影響 |
|-----------|------|
| 引継ぎメモの作成に時間がかかる | FSの生産性低下、残業増加 |
| 内容が薄い・不足している | 役務の対応品質低下 |
| FSによって品質にばらつきがある | 組織としての品質担保が困難 |
| 商談で得た情報が忘れられる | 貴重な情報の損失 |

### 1.3 期待する効果

| 効果 | 定量目標（案） |
|------|--------------|
| 作成時間の短縮 | 30分 → 5分（83%削減） |
| 品質の向上 | 品質スコア平均70点以上 |
| 品質の均一化 | ばらつき（標準偏差）50%削減 |
| 情報の網羅性向上 | 必須項目充足率95%以上 |

## 2. ユーザーストーリー

### 2.1 メインユーザー
**FS（フィールドセールス）**

### 2.2 ユーザーストーリー

```
FSとして、
商談の録音データや会話メモを入力すると、
引継ぎメモが自動生成されてほしい。
なぜなら、引継ぎメモの作成に時間がかかり、
内容も抜け漏れが多いから。
```

### 2.3 利用シーン

| シーン | 入力データ | 期待する出力 |
|--------|-----------|-------------|
| 商談直後 | Zoom/電話の文字起こし | 6項目の引継ぎメモ下書き |
| 複数回商談後 | 複数回分の会話記録 | 統合された引継ぎメモ |
| 成約時 | 全商談記録＋契約情報 | 完成版引継ぎメモ |

## 4. 技術要件

### 4.1 アーキテクチャ選択肢

| 方式 | 概要 | メリット | デメリット |
|------|------|---------|-----------|
| **A. プロンプトエンジニアリング** | Claude APIに詳細プロンプトを渡す | 開発コスト低、即座に開始可能 | プロンプト肥大化、一貫性に課題 |
| **B. RAG方式** | 良い例をベクトルDBに格納し検索 | 品質安定、例示による学習 | DB構築コスト、検索精度依存 |
| **C. Fine-tuning** | 専用モデルを学習 | 最高品質、一貫性高い | コスト高、学習データ必要 |
| **D. ハイブリッド** | A + B の組み合わせ | バランス良い | 中程度の開発コスト |

### 4.2 推奨アプローチ

**Phase 1: プロンプトエンジニアリング（即座に開始）**
- 作成したドキュメントをシステムプロンプトに組み込む
- 低コストで効果検証可能

**Phase 2: RAG方式の追加（効果検証後）**
- 良い例をベクトルDBに格納
- 類似案件の引継ぎ例を参照して品質向上

**Phase 3: 継続的改善**
- フィードバックに基づくプロンプト改善
- 良い例の継続的な追加

### 4.3 技術スタック（案）

| コンポーネント | 選択肢 | 備考 |
|---------------|-------|------|
| LLM | Claude API（Sonnet/Opus） | 日本語品質、長文対応 |
| UI | Streamlit / Gradio | 素早いプロトタイプ |
| ベクトルDB（Phase 2） | Pinecone / Chroma | RAG用 |
| 音声文字起こし（将来） | Whisper API | 音声入力対応時 |

## 12. Zoom Phone連携（電話文字起こし自動取得）

### 12.1 課題
- 商談はZoom会議で文字起こし取得可能
- 商談後の電話フォローは文字起こしが取れない
- 電話での重要な会話が引継ぎメモに反映されない

### 12.2 解決策: Zoom Phone文字起こし連携

```
┌─────────────────────────────────────────────────────────────┐
│                    データフロー                              │
├─────────────────────────────────────────────────────────────┤
│                                                             │
│  Salesforce                    Zoom Phone                   │
│  ┌─────────────┐              ┌─────────────┐              │
│  │ Project__c  │              │ 通話履歴    │              │
│  │ Opportunity │              │ 文字起こし  │              │
│  │     ↓       │              │     ↑       │              │
│  │ Contact     │──電話番号で──→│   検索      │              │
│  │ (電話番号)  │    紐付け    │             │              │
│  └─────────────┘              └─────────────┘              │
│         │                            │                      │
│         └──────────┬─────────────────┘                      │
│                    ↓                                        │
│         ┌─────────────────────┐                            │
│         │ 統合データ          │                            │
│         │ ・商談文字起こし    │                            │
│         │ ・電話文字起こし    │                            │
│         └──────────┬──────────┘                            │
│                    ↓                                        │
│         ┌─────────────────────┐                            │
│         │ AI引継ぎメモ生成    │                            │
│         └─────────────────────┘                            │
│                                                             │
└─────────────────────────────────────────────────────────────┘
```

### 12.3 Zoom Phone API連携方式

| ステップ | 処理内容 | 使用API/データ |
|---------|---------|---------------|
| 1 | Salesforceから電話番号取得 | Contact.Phone, Contact.MobilePhone |
| 2 | 関連するProject/Opportunityを特定 | Project__c, Opportunity |
| 3 | Zoom Phoneで該当電話番号の通話検索 | Zoom Phone API: /phone/call_history |
| 4 | 通話録音の文字起こし取得 | Zoom Phone API: /phone/recordings/{id}/transcript |
| 5 | 期間でフィルタ（商談期間内） | 商談開始日〜成約日 |
| 6 | 文字起こしを統合 | 商談＋電話の全テキスト |

### 12.4 Zoom Phone API エンドポイント（確認済み）

| No | 用途 | メソッド | エンドポイント |
|----|------|---------|---------------|

#### 処理フロー
```
Step 1: 通話ログ一覧取得（期間＋電話番号で絞り込み）
  GET /accounts/{accountId}/phone/call_logs
  ?start_time={from}&end_time={to}
        ↓
Step 2: 通話ログから録音IDを取得
  GET /accounts/{accountId}/phone/call_logs/{callLogId}/recordings
        ↓
Step 3: 録音IDで文字起こしダウンロード
  GET /accounts/{accountId}/phone/recording_transcript/download/{recordingId}
        ↓
Step 4: (任意) AI通話サマリーも取得
  GET /accounts/{accountId}/phone/user/{userId}/ai_call_summary/{aiCallSummaryId}
```

#### 認証方式
- **Server-to-Server OAuth** (推奨)
- 必要スコープ: `phone:read:admin`, `recording:read:admin`

### 12.5 実装アーキテクチャ

```python
# 概念コード（実装時のベース）

import requests
from datetime import datetime

class ZoomPhoneClient:
    """Zoom Phone API クライアント"""

    BASE_URL = "https://api.zoom.us/v2"

    def __init__(self, account_id, client_id, client_secret):
        self.account_id = account_id
        self.client_id = client_id
        self.client_secret = client_secret
        self.access_token = None

    def authenticate(self):
        """Server-to-Server OAuthトークン取得"""
        url = "https://zoom.us/oauth/token"
        params = {
            "grant_type": "account_credentials",
            "account_id": self.account_id
        }
        response = requests.post(url, params=params,
                                 auth=(self.client_id, self.client_secret))
        self.access_token = response.json()["access_token"]

    def _headers(self):
        return {"Authorization": f"Bearer {self.access_token}"}

    def get_call_logs(self, user_id=None, start_time=None, end_time=None):
        """通話ログ一覧取得"""
        if user_id:
            url = f"{self.BASE_URL}/accounts/{self.account_id}/phone/users/{user_id}/call_logs"
        else:
            url = f"{self.BASE_URL}/accounts/{self.account_id}/phone/call_logs"
        params = {}
        if start_time:
            params["start_time"] = start_time
        if end_time:
            params["end_time"] = end_time
        return requests.get(url, headers=self._headers(), params=params).json()

    def get_recordings_for_call(self, call_log_id):
        """通話ログの録音取得"""
        url = f"{self.BASE_URL}/accounts/{self.account_id}/phone/call_logs/{call_log_id}/recordings"
        return requests.get(url, headers=self._headers()).json()

    def download_transcript(self, recording_id):
        """文字起こしダウンロード"""
        url = f"{self.BASE_URL}/accounts/{self.account_id}/phone/recording_transcript/download/{recording_id}"
        return requests.get(url, headers=self._headers()).text


def get_phone_transcripts_for_project(sf_service, zoom_client, project_id):
    """
    プロジェクトに関連する電話文字起こしを取得

    フロー:
    1. Salesforce → 関連Contactの電話番号取得
    2. Salesforce → プロジェクト期間取得
    3. Zoom Phone → 該当期間の通話ログ検索
    4. Zoom Phone → 録音ID取得 → 文字起こしDL
    """

    # 1. Salesforceから関連Contactの電話番号を取得
    contact_query = f"""
        SELECT Contact__r.Phone, Contact__r.MobilePhone,
               Contact__r.Name
        FROM Project__c
        WHERE Id = '{project_id}'
    """
    contacts = sf_service.run_query(contact_query)
    phone_numbers = normalize_phone_numbers(contacts)

    # 2. プロジェクト期間を取得
    project_query = f"""
        SELECT CreatedDate, LastHandoverDate__c
        FROM Project__c
        WHERE Id = '{project_id}'
    """
    project = sf_service.run_query(project_query)
    start_date = project['records'][0]['CreatedDate']
    end_date = project['records'][0]['LastHandoverDate__c'] or datetime.now().isoformat()

    # 3. Zoom Phoneから通話ログ取得
    call_logs = zoom_client.get_call_logs(
        start_time=start_date,
        end_time=end_date
    )

    # 4. 電話番号でフィルタ → 録音 → 文字起こし
    transcripts = []
    for log in call_logs.get('call_logs', []):
        caller = normalize_phone(log.get('from', ''))
        callee = normalize_phone(log.get('to', ''))

        if caller in phone_numbers or callee in phone_numbers:
            # 録音取得
            recordings = zoom_client.get_recordings_for_call(log['id'])

            for rec in recordings.get('recordings', []):
                transcript = zoom_client.download_transcript(rec['id'])
                transcripts.append({
                    'date': log['start_time'],
                    'duration': log.get('duration', 0),
                    'direction': 'inbound' if callee in phone_numbers else 'outbound',
                    'caller': log.get('from'),
                    'callee': log.get('to'),
                    'transcript': transcript
                })

    return transcripts


def normalize_phone(phone):
    """電話番号の正規化（日本形式）"""
    if not phone:
        return ''
    phone = phone.replace('-', '').replace(' ', '').replace('+81', '0')
    return phone[-10:]  # 末尾10桁で比較
```
