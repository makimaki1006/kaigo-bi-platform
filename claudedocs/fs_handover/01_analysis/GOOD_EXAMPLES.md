# FS引継ぎメモ 良い例の実例集

## 概要

実際のプロジェクトから抽出した良い例をまとめています。
これらをテンプレートとして活用してください。

---

## 1. 募集職種の良い例

### 例1: 社会福祉法人清香会

```
【正職員】介護職　3名
【概要】
正社員×デイ1名、特養夜勤なし1名、特養夜勤あり1名
【求める人材】
・資格保持　※介福優遇
・パソコンに抵抗がない方
・異業種経験者でもご紹介OK
【初任給】
正社員月給19.4万円～
昇給/年1回
賞与/年3回
【補足】
・残業少なめ
・シフト：デイは早番遅番あり、特養は夜勤あり/なし選択可
```

**ポイント:**
- 雇用形態・人数が明確
- 求める人材像が具体的
- 給与・待遇も記載
- シフトの柔軟性も言及

## 2. 採用課題の良い例

### 例1: 社会福祉法人ざおう福祉会

```
【採用背景】
2024年4月に新施設オープン予定。
既存施設からの異動もあるが、新規で介護職10名程度必要。

【これまでの採用活動】
・ハローワーク：応募少ない
・Indeed：応募はあるが辞退多い
・地元の人材紹介会社：1社利用中だが紹介数少ない

【課題】
1. 地方のため母集団が少ない
2. 給与水準が県内平均並みで差別化しにくい
3. 新施設の認知度が低い

【採用要件】
・人数：10名
・期限：2024年3月末まで
・緊急度：高（オープン日が決まっている）

【決裁フロー】
施設長が最終決裁。面接は施設長＋介護主任の2名体制。
```

**ポイント:**
- 背景が具体的（新施設オープン）
- これまでの取り組みと結果が明記
- 課題が箇条書きで整理
- 決裁フローも記載

## 3. 導入ポイントの良い例

### 例1: 株式会社イデアル

```
Q: なぜフォーラムエンジニアリングを選んだ？
A: 「価格が他社より安かったのと、営業の〇〇さんの対応が丁寧だったから」

【評価されたポイント】
・価格：他社見積もり3社の中で最安値（約15%安い）
・対応：見積もり依頼から3日以内に提案、質問への回答も早い
・提案内容：うちの施設に合った求人票の書き方を提案してくれた

【競合との比較】
・A社：価格高い、対応遅い
・B社：価格は同等だがサービス内容が不明瞭
→ 当社は価格＋対応スピード＋提案力で勝利

【懸念と解消】
・懸念：「本当に採用できるのか？過去に紹介会社で失敗している」
・解消：成功事例（同業他社での採用実績）を説明、初月保証の説明
```

**ポイント:**
- 先方の言葉をそのまま引用
- 競合比較が具体的
- 懸念と解消方法がセット

## 4. 刺さったポイントの良い例

### 例1: 株式会社コンフォートプレイス

```
【商談で響いたポイント】

①「御社の求人票、もったいないです」
  → 現状の求人票を見せてもらい、改善点を3つ具体的に指摘
  → 「そういう視点はなかった」と驚かれた

②採用成功事例の紹介
  → 同じ地域・同規模の介護施設で、当社経由で3名採用できた事例
  → 「具体的でイメージしやすい」と評価

③初期費用ゼロのリスクフリー提案
  → 成功報酬型であることを強調
  → 「試しやすい」と決め手に

【先方の課題と解決策提示】
・課題：「応募が来ない」
・解決策：ターゲット層を絞り込んだ求人票改善＋SNS広告の提案
・評価：「今までの紹介会社は紹介するだけだった」
```

**ポイント:**
- 具体的な提案内容と反応がセット
- 先方の言葉を引用
- 競合との違いが明確

## 5. 先方関係者情報の良い例

### 例1: 株式会社イデアル

```
【採用担当者】
・氏名：山田花子
・役職：人事部主任
・連絡先：090-XXXX-XXXX（携帯優先）
・人柄：明るくテキパキしている。結論から話すタイプ。
・連絡の好み：電話＞メール。午前中が繋がりやすい。

【決裁者】
・氏名：鈴木太郎
・役職：施設長
・関与度：最終面接に同席。採用可否は施設長が決定。
・特徴：じっくり考えるタイプ。即断即決しない。

【現場責任者】
・氏名：田中次郎
・役職：介護主任
・面接での重視点：人柄、チームワーク、体力
・特徴：現場のことを一番分かっている。候補者との相性を重視。

【連絡フロー】
基本は山田さんに連絡 → 重要な件は鈴木施設長にCC
面接調整は山田さん経由で田中主任と調整
```

**ポイント:**
- 3者の役割が明確
- 人柄・コミュニケーションスタイルが具体的
- 連絡フローも記載

## 6. 担当者注意点の良い例

### 例1: 社会福祉法人ざおう福祉会

```
【連絡について】
・電話がベスト（メールは見落とし多い）
・午前中は会議が多いため、13:00-17:00が繋がりやすい
・金曜日は現場に出ていることが多い

【コミュニケーションスタイル】
・結論から話す方が好まれる
・データや数字で説明すると納得しやすい
・雑談は好まない（忙しいアピールあり）

【避けるべき話題】
・前任の人材紹介会社の話（トラブルあった様子）
・他社との比較をあからさまにする

【重視していること】
・候補者の人柄（スキルより優先）
・長く働いてくれるか
・地域への愛着

【特別対応】
・施設長は耳が遠いため、ゆっくり大きな声で話す
・面接は必ず施設見学とセット（候補者のミスマッチ防止のため）
```

**ポイント:**
- 連絡のタイミングが具体的
- NG話題が明記
- 特別対応も漏れなく記載

## ダメな例（避けるべき記載）

### NG例1: 内容なし
```
【募集職種】
介護職
```
→ 何も情報がない

### NG例2: テンプレートのみ
```
【採用課題】
・採用背景：
・これまでの活動：
・課題：
```
→ 項目だけで中身がない

### NG例3: 投げやり
```
【刺さったポイント】
特になし
```
→ 成約したのに刺さったポイントがないはずがない

### NG例4: 抽象的
```
【先方関係者情報】
担当者：田中さん
いい人です
```
→ 情報として役に立たない
