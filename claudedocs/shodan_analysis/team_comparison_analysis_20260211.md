# 営業チーム行動指標比較分析レポート

**分析日**: 2026-02-11
**対象期間**: 2024年前半（1-6月） vs 2026年1月
**対象**: セールスメンバーのみ（コンサル3名 小幡・丹司・遠藤は除外）
**分析商談数**: 2024H1: 305件 / 2026Jan: 308件（合計613件）

---

## 1. エグゼクティブサマリー

### 最重要発見: 「組織全体の劣化」と「個人差の拡大」

チーム全体で商談行動パターンが変化しているが、**全員が同じように劣化しているわけではない**。
むしろ**適応できた者とできなかった者の差が拡大**している。

| 発見 | 意味 |
|------|------|
| 顧客相槌率: 8名中7名が低下 | 市場環境の変化（顧客側の変化）を示唆 |
| クロージングKW: **▼49.6%** | 組織として「クロージング武器」が枯渇 |
| データKW/分がトップ/ボトムの最大差別化要因 | 両期間とも「数字で語れるか」が分水嶺 |
| 松風・灰野は適応・改善 | 個人の学習能力で差がついている |
| 佐藤は最大劣化 | かつての強み（圧倒力）が完全に機能停止 |

1. **独白が57%減**: 長い説明ブロック（268→115文字）が激減。顧客の集中力低下 or プレゼン力低下
2. **顧客相槌率の全体低下（-20%）**: 顧客が「うん、うん」と聞かなくなった → 商材への新鮮味低下
3. **クロージングKW半減（-50%）**: キャンペーン・限定・割引等の武器が組織として供給されていない
4. **P質問減少（-21%）**: 課題深掘りの質問力が組織全体で低下

### 4-3. 勝ちパターンの進化

```
2024H1の勝ちパターン:
  データ武装(+73%) + 会話支配(+17%) + クロージング(+97%)
  → 「圧倒して、畳みかけて、刈り取る」

2026Janの勝ちパターン:
  データ武装(+71%) + 冒頭攻勢(+25%) + 課題深掘り(+38%)
  → 「冒頭で掴み、データで説得し、課題に寄り添う」
```

**変わったこと**:
- **CloseKWの差が消滅**（+97% → +4%）: クロージング武器は全体として枯渇し、差別化要因ではなくなった
- **冒頭3分Qが新たな差別化要因に**: 最初の掴みが勝敗を分ける
- **数字/分が差別化要因として浮上**: 具体的な数字を使えるかどうか

**変わらないこと**:
- **データKW/分は両期間とも最大の差別化要因**（+73% → +71%）

## 6. 組織全体で起きていること

### 6-1. 全員に共通する変化（= 環境要因）

| 変化 | 傾向 | 原因推定 |

| **顧客相槌率の低下** | 8名中7名が低下（-10%〜-35%） | 顧客リテラシー向上、商材への新鮮味低下 |

| **独白長の短縮** | 全体-57% | 長い説明が通用しなくなった |

→ これらは**個人のスキルではなく、市場環境・組織施策の変化**が原因

## 8. 提言

### P0: 即座に実行（組織施策）

1. **データアップデートの仕組み化**
   - 全メンバーの相槌率が低下 = 同じ話が通用しなくなった
   - 月次でデータセット（市場データ、成功事例、競合情報）を更新・配布
   - **基準値**: データKW/分 ≧ 1.5 を全員の目標に

2. **クロージング武器の供給**
   - CloseKW -50%は組織的な武器枯渇
   - 期間限定キャンペーン、顧客事例、ROI実績等を定期供給

### P1: 1ヶ月以内（個人指導）

3. **佐藤丈太郎への集中介入**
   - かつてのトップ営業が最大劣化。放置すると離職リスク
   - 志村の「数字強化+CloseKW維持」パターンをモデルに
   - **目標**: データKW/分を0.73→1.3以上に回復

4. **嶋谷・小林のデータ武装**
   - 商談件数は確保できているが質的指標が弱い
   - **目標**: データKW/分を1.0→1.5に引き上げ

5. **清飛羅の冒頭3分Q改善**
   - 全指標が緩やかに低下。冒頭3分Q -28%が特に問題
   - 松風の冒頭攻勢パターンを参考に

### P2: 3ヶ月以内（組織改善）

6. **松風モデルの横展開**
   - 冒頭3分Q +86%、P質問+113% → 最も成功した適応モデル
   - 「質問主体の商談設計」をチーム標準に

7. **志村モデルのベストプラクティス化**
   - 唯一CloseKWが増加し、数字/分+63%
   - 「数字+クロージング」の具体的トーク例を共有

8. **冒頭3分の標準化**
   - 冒頭3分Qが新たなトップ/ボトム差別化要因（+25%）
   - 商談冒頭のテンプレート（最低3つの質問）を策定

### 指標定義

| 指標 | 定義 |
|------|------|
| 文字比率 | 営業の発言文字数 / 全体文字数 |
| 数字/分 | 1分あたりの数字出現回数 |
| データKW/分 | 1分あたりのデータ関連KW出現回数（調査/人口/市場/業界/統計/実績/データ/割合等） |
| 冒頭3分Q | 最初3分間の営業の質問回数 |
| P質問 | 課題質問（「課題」「問題」「困」「悩」「難しい」等を含む質問） |
| 顧客相槌率 | 顧客の短い発話（≦10文字）の割合 |
| CloseKW | クロージング関連KW合計（キャンペーン/限定/割引/特別/申込/契約/決め/早め等） |
