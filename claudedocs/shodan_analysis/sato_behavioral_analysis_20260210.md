## 1. エグゼクティブサマリー

### 最重要発見: 「会話支配率の崩壊」

**イケイケ期の佐藤は会話の85%を占有する「一方的ピッチ型営業」で勝っていた。**
だめだめ期はその支配力が67%に低下し、顧客が主導権を握り始めた。

一見すると「顧客が多く話す＝良い営業」に見えるが、実態は逆。
佐藤の「武器」であるデータ殴打（数字使用頻度）が40%減少した結果、
顧客を圧倒できず、懐疑的な質問に振り回される構図に転落している。

## 3. 構造的仮説

### 仮説A: 「支配型営業」の崩壊

イケイケ期の勝ちパターンは以下の構造だった：

```
冒頭3分: 質問5回で状況把握
  ↓
独白ピッチ: 数字+データで畳み掛け（85%支配、0.72数字/分）
  ↓
顧客: 受け身（48%相槌）→ 圧倒されて成約
```

だめだめ期はこの構造が崩れた：

```
冒頭3分: 質問3.9回（▼23%）→ 状況把握が甘い
  ↓
独白ピッチ: 短く弱い（67%支配、0.43数字/分）
  ↓
顧客: 能動化（質問+53%、長い発話+133%）→ 主導権を奪われる
  ↓
佐藤: 顧客の質問に振り回され、クロージングできず
```

### 仮説B: 数字ストックの枯渇

- 数字使用頻度▼40%は「知識不足」ではなく「ストックの陳腐化」の可能性
- イケイケ期に使っていた市場データ・競合情報が1.5年経過で古くなった
- 新しいデータを仕入れていないため、数字を出せない → 抽象表現に逃げる

### 仮説C: 再商談のジレンマ

- だめだめ期は71%が再商談（過去失注先のリサイクル）
- 再商談相手は一度ピッチを聞いているため、同じ独白が通用しない
- 顧客側も「前回と何が違うの？」という目線で能動的に質問する
- → 新規相手には通用した支配型営業が、再商談では機能しない

## 11. 統合的結論: 「佐藤営業モデル」の構造と崩壊メカニズム

### イケイケ期の「勝ちの方程式」

```
Step 1: 冒頭3分 → 質問5回でヒアリング（79%支配）
Step 2: 5-10分 → データ殴打開始（83%支配）
  - 「人口」「市場」「調査」「競合」「施設」データを投入
  - 「エリア」の名前を出して具体性を担保
Step 3: 15-25分 → 圧倒ゾーン（90%超支配）
  - 「させていただ」63回の丁寧語で畳み掛け
  - 「人事部」のキラーフレーズで再定義
  - 独白200文字超を23回繰り出す
Step 4: 35-50分 → クロージング
  - 「キャンペーン」「割引」で今決める理由
  - 「損失」「機会」で機会損失を突きつけ

→ 顧客は48%が相槌 = 受け身のまま成約
```

### だめだめ期の「負けの構造」

```
Step 1: 冒頭3分 → 質問3.9回（▼23%）→ ヒアリングが浅い
Step 2: 5-10分 → データ殴打が始まらない（58%支配）
  - 「人口」▼84%、「調査」▼76%、「市場」▼67%
  - 数字の代わりに「効果」「実績」という抽象語
Step 3: 15-25分 → 圧倒ゾーンに入れない（68-71%）
  - 「人事部」▼92%、キラーフレーズなし
  - 独白が短い（118文字 vs 195文字）
Step 4: 35-50分 → クロージング武器なし
  - 「キャンペーン」▼76%、「割引」▼83%
  - 「今決める理由」を提示できない

→ 顧客が主導権を握り（質問+53%、長い発話+133%）
→ 「ちょっと検討します」で終了
```

### 崩壊の因果関係

```
根本原因: データストック（市場調査・人口動態・競合情報）の陳腐化
    ↓
直接原因: データ殴打ができない → 数字使用▼40%
    ↓
連鎖反応:
  ├→ 独白が短くなる（▼39%）→ 圧倒できない
  ├→ 抽象表現が増える（+105-157%）→ 説得力がない
  ├→ キラーフレーズ消滅（人事部▼92%）→ 再定義できない
  └→ クロージング武器消滅（割引▼83%）→ 今決める理由がない
    ↓
結果: 顧客が主導権を握る（質問+53%）
    ↓
最終結果: 成約率0.0%
```
