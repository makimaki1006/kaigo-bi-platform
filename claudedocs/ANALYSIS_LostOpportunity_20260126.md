**分析対象期間**: 2025年4月1日 ～ 2026年1月15日（実績データ）

## エグゼクティブサマリー

| 指標 | 値 |
|------|-----|
| 総商談数（実績期間） | 約4,300件 |
| 受注 | 約430件 |
| 失注 | **3,864件** |

### 主要な発見

1. **失注の28%が3つの理由に集約**: 導入予定なし(11.2%) + 理解不足(8.4%) + 金額懸念(8.0%)
2. **2025年4月にデータ品質が劇的向上**: Zoom連携率30%→97%、「その他」64%→3%
3. **決裁権者に会えると成約率3倍**: 21.4% vs 7.4%
4. **Zoom分析**: 顧客のコミットメント発言が受注と強く相関（+89%）

→ 「その他」が64%→3%に激減、具体的な失注理由が記録されるようになった

### 1.4 最終到達フェーズの変化

| フェーズ | 4月以前 | 4月以降 | 差分 |
|---------|--------|--------|------|
| **01フェーズ** | **68.9%** | **36.6%** | **-32.4%** |
| **02フェーズ** | 16.9% | **43.9%** | **+26.9%** |
| 03フェーズ | 10.6% | 15.9% | +5.3% |

→ 初回商談での失注が減少し、2回目以降に進むようになった

### 1.5 総括: 2025年4月に何が起きたか

1. **プロセス変更**: Zoom商談記録の義務化（30%→97%）
2. **データ品質向上**: 失注理由「その他」の激減（64%→3%）
3. **商談プロセス変化**: 初回失注の減少（69%→37%）
4. **成約率の見かけ上の低下**: 全体14.0%→9.6%

**解釈**:
- 4月以前は「記録されない失注」が多かった（Zoom連携30%、その他64%）
- 4月以降は全商談が正確に記録されるようになり、**実態が見えるようになった**
- 成約率の低下は「悪化」ではなく「**正確な計測の開始**」と解釈すべき
