## エグゼクティブサマリー

### 主要発見

| 項目 | 発見 |
|------|------|
| **主因** | 営業スキル（懸念払拭力）の低下 |
| **最大の変化** | 「投資懸念」が+13.1pt増加 |
| **影響範囲** | 初回商談・再商談ともに同様の傾向 |
| **特定人物** | 5名が+10pt以上悪化 |
| **フェーズ** | 代表者商談（決裁者フェーズ）で+10.1pt悪化 |

### 結論

```
成約率低下の真因 = 「投資懸念」の払拭ができていない

特に:
- 代表者商談（決裁者フェーズ）での懸念払拭力不足
- 特定の営業人員での大幅悪化
- 介護（高齢者）施設での悪化が顕著
```

## 1. 仮説検証

### 検証した仮説

| 仮説 | 結果 | 詳細 |
|------|------|------|
| 再商談の割合が増えている | ✅ 支持 | 17.6% → 24.5%（+7.0pt） |
| 再商談だから懸念系失注が多い | ❌ 不支持 | 初回21.0% vs 再商談21.0%（差なし） |
| 初回商談でも懸念系が増えている | ✅ 支持 | 13.5% → 21.0%（+7.5pt） |

### 注意事項

- 再商談率の月別変動は**退職者ローラー**の影響
- 時系列トレンドとして解釈しない

## 6. 構造的問題の整理

```
【表面的な現象】
・懸念系失注率: 14.0% → 21.0%（+7.0pt）
・成約率低下

【構造的な問題】
1. 営業スキル問題
   - 投資懸念の払拭ができていない
   - 代表者商談（決裁者フェーズ）で特に顕著
   - 特定人物での大幅悪化

2. アポ供給問題（副因）
   - 再商談率の増加（+7.0pt）
   - 退職者ローラーの影響

3. 案件特性
   - 介護（高齢者）施設で悪化
   - 短期商談で悪化
```
