# 有効求人倍率 Salesforce連携レポート

**作成日**: 2026-01-21
**ステータス**: Account完了 / JobCard検証完了

---

## 1. 概要

MECEマッピングデータ（1,918市区町村）を使用して、SalesforceのAccount・JobCard__cオブジェクトに有効求人倍率を付与する作業を実施。

### データソース

| ファイル | 内容 |
|---------|------|
| `data/job_openings_ratio/complete_mece_municipality_hellowork_mapping.csv` | 市区町村×ハローワーク×求人倍率マッピング |

### 求人倍率データカバレッジ

| 項目 | 件数 |
|------|------|
| 総市区町村数 | 1,918件 |
| 求人倍率あり | 1,755件（91.5%） |
| 求人倍率なし | 163件（8.5%） |

## 2. Account.Description 更新【完了】

### 2.1 処理概要

| 項目 | 値 |
|------|-----|

| 更新フィールド | Description（説明） |

```
Address__c（住所）から市区町村を抽出

```

## 3. JobCard__c（求人）突合検証【完了】

### 3.1 フィールド構成

| フィールド | API名 | 型 |
|-----------|------|-----|
| 都道府県 | `Prefecture__c` | picklist |
| 市区町村 | `City__c` | string |

### 3.2 マッチング方法

```
Prefecture__c + City__c を結合
  ↓
郡名除去等の正規化（例: 駿東郡長泉町 → 長泉町）
  ↓
MECEマッピングと突合
```

### 3.3 検証結果

| カテゴリ | 件数 | 割合 | 備考 |
|---------|------|------|------|
| 総件数 | 10,700件 | 100% | |
| **マッチ成功（倍率あり）** | **6,410件** | **59.9%** | 付与可能 |
| マッチ成功（倍率なし） | 633件 | 5.9% | データ欠損地域 |
| マッチ失敗 | 796件 | 7.4% | データ品質問題 |
| データなし | 2,861件 | 26.7% | フィールド空 |

### 3.4 突合不可レコードの内訳

#### A. 「////」含むレコード（スキップ対象）

| 項目 | 件数 |
|------|------|
| 「////」含む | 1,100件 |
| 特徴 | 全て都道府県or市区町村が空 |

サンプル：
```
【////】こども発達支援ルームピッピーノ
【////】医療法人梓誠会
```

#### B. マッチ成功だが倍率データなし（633件）

**県全域で非公表は山梨県のみ**。他は特定市区町村のデータ欠損。

| 都道府県 + 市区町村 | 件数 | 理由 |
|-------------------|------|------|
| 長崎県 + 長崎市 | 72件 | 長崎市のみデータなし |
| 京都府 + 京都市伏見区 | 44件 | 伏見区のみデータなし |
| 鹿児島県 + 鹿児島市 | 39件 | 鹿児島市のみデータなし |
| 静岡県 + 駿東郡長泉町 | 36件 | 長泉町のみデータなし |
| 岡山県 + 岡山市北区 | 35件 | 岡山市のみデータなし |
| 福島県 + 郡山市 | 20件 | 郡山市のみデータなし |
| 千葉県 + 千葉市各区 | 58件 | 千葉市のみデータなし |
| 北海道 + 苫小牧市 | 16件 | 苫小牧市のみデータなし |
| 山梨県 + 各市町村 | 26件 | **県全域で非公表** |

#### C. マッチ失敗（796件）⚠️ データ品質問題

**都道府県と市区町村の組み合わせが不正**

| 例 | 問題 |
|----|------|
| 神奈川県 + 長岡京市 | 長岡京市は京都府 |
| 神奈川県 + 旭川市 | 旭川市は北海道 |
| 大阪府 + 神戸市垂水区 | 神戸市は兵庫県 |
| 東京都 + 富士市 | 富士市は静岡県 |
| 福岡県 + 金沢市 | 金沢市は石川県 |

→ Prefecture__cフィールドに誤った値が入っている

#### D. 市区町村が空（1,761件）

| 内訳 | 件数 |
|------|------|
| 都道府県のみ（市区町村空） | 1,759件 |
| 両方空 | 2件 |
