# FY2026 4月 営業活動 戦略運用方針

**作成日**: 2026-02-24
**データ期間**: FY2025（2025年4月〜2026年1月）実績ベース
**対象**: 新規開拓（初回商談）のみ ※解約後再商談は除外

---

## 1. 全体像：FY2025の実態

### 商談構成

| 区分 | 件数 | 構成比 | 受注率 |
|------|------|--------|--------|
| **新規商談** | 2,898件 | 79.5% | **9.1%** |
| 失注後再商談 | 664件 | 18.2% | 4.5% |
| 解約後再商談 | 84件 | 2.3% | 54.8%（対象外） |

**重要な発見**:
- 失注後再商談（4.5%）は新規商談（9.1%）の半分の受注率。**再商談は効率が悪い。新規に注力すべき。**
- ただし例外あり: 失注3回以上の再商談は受注率9.1%まで回復（粘り強い先は成約する）
- 訪問リハビリテーションは再商談（10.0%）が新規（7.8%）を上回る唯一のサービス種別

### 施設形態別受注率（新規のみ）

| 施設形態 | 件数 | 受注率 | 判定 |
|---------|------|--------|------|
| **保育** | 43件 | **27.9%** | **最優先** |
| **介護（高齢者）** | 281件 | 15.7% | **注力** |
| 医療 | 129件 | 3.1% | 要選別 |
| 障がい福祉 | - | 1〜10% | 要選別 |
| 不明（業種未特定） | 426件 | 0.2% | **絶対回避** |

## 3. 最重要発見：訪問看護の二面性

**同じ「訪問看護」でも、運営法人の種類で受注率が15倍違う。**

| セグメント | 件数 | 受注率 | 判定 |
|-----------|------|--------|------|
| **介護法人の訪問看護（新規）** | 33件 | **45.5%** | **最優先ターゲット** |
| 医療法人の訪問看護（新規） | 129件 | 3.1% | 回避推奨 |
| 障がい福祉の訪問看護 | 21件 | 0.0% | 絶対回避 |

**見分け方**: Account.IndustryCategory__c（業界カテゴリー）
- 「介護」を含む → 介護法人 → **積極対応**
- 「医療」を含む → 医療法人 → 回避

**Salesforceレポート**:

## 5. 回避ルール適用による改善シミュレーション

### 通年ベース

| 回避ルール | 回避件数 | 残り件数 | 残り受注率 | 改善幅 |
|-----------|---------|---------|-----------|--------|
| 何もしない | 0 | 3,525 | 8.1% | - |
| 業種不明のみ回避 | 426 | 3,099 | 9.1% | +1.1pt |
| +障がい福祉中規模 | 639 | 2,886 | 9.7% | +1.6pt |
| +医療大規模 | 812 | 2,713 | 10.2% | +2.1pt |
| **全ルール適用** | **1,057** | **2,468** | **11.0%** | **+2.9pt** |

**全ルール適用時**: 商談の30%を回避 → 残り70%の受注率が **8.1% → 11.0%** に改善

### 訪問看護の選別を加えると

医療法人の訪問看護129件（受注率3.1%）を回避し、介護法人の訪問看護33件（受注率45.5%）に集中すると、訪問看護セグメントだけで受注率10.7% → 45.5%に改善。

## 6. 4月の具体的アクションプラン

### Week 1-2（4月1日〜14日）

1. **アタックリストの再構成**
   - Tier C のセグメントをリストから除外
   - 業種不明（426件分）を最優先で除外
   - 訪問看護は Account.IndustryCategory__c で介護/医療を判別

### Week 3-4（4月15日〜30日）

2. **注力セグメントへの集中投下**
   - 保育全般 → 通年好調（27.9%）、4月も安定
   - 介護法人×訪問看護 → 最高受注率（45.5%）
   - 介護×有料老人ホーム → 高受注率（40.0%）

3. **失注後再商談の抑制**
   - 再商談枠を削減し、新規商談枠を拡大
   - 再商談（4.5%）→ 新規（9.1%）へのリソースシフト

## 8. 判定フローチャート（営業担当向け）

```
新規リードが来たら：

1. 業種情報はあるか？
   ├─ NO → ❌ 回避（受注率0.2%）
   └─ YES ↓

2. 施設形態は？
   ├─ 保育 → ✅ 最優先（27.9%）
   ├─ 介護 ↓
   │   ├─ 訪問看護？
   │   │   ├─ 介護法人 → ✅ 最優先（45.5%）
   │   │   └─ 医療法人 → ❌ 回避（3.1%）
   │   ├─ 従業員1-10人？ → △ 優先度下げ（4.7%）
   │   └─ その他 → ✅ 対応（10-16%）
   ├─ 医療 ↓
   │   ├─ 301人+（大病院）→ ❌ 回避（2.9%）
   │   ├─ 11-30人 → ✅ 積極対応（14.1%）
   │   └─ その他 → ○ 通常対応
   └─ 障がい福祉 ↓
       ├─ 1-10人 or 不明 → ○ 対応OK（9.6%）
       └─ 11人以上 → ❌ 回避（1〜4%）

3. 4月限定チェック：
   ├─ クリニック → △ 4月は慎重に（通年28.6%→4月6.9%）
   ├─ 特養 → △ 4月は慎重に（通年11.1%→4月3.6%）
   └─ デイサービス/介護付き有料 → △ 4月は慎重に（通年8-10%→4月0%）
```
