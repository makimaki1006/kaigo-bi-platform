### 主要な発見

1. **全体ではパートの採用単価が正社員より約21万円高い**が、これは地域構成比の違いによるシンプソンのパラドックス
2. **首都圏の正社員が最も採用困難**（単価¥155万/404日）
3. **関西圏・地方ではパートが困難**（単価¥92〜110万/222〜342日）
4. **採用実績があるPJは全体の約10%**（17/178）。残り90%のPJは未採用で総投資額¥1.3億
5. **ジョブメドレーオプション**: 4PJすべて未採用 → 効果測定不可
6. **「はるいろ訪問看護ステーション」（福岡）が正社員採用の53%（8/15名）を占め**、全体平均を大きく左右

## 14. 分析手法・MECE検証

### 14.1 採用単価の計算方法

```
採用単価 = プロジェクト配分コスト ÷ 採用数

プロジェクト配分コスト = プロジェクト総コスト × 配分比率
配分比率 = 該当雇用形態の求人数 ÷ プロジェクト全求人数

プロジェクト総コスト（優先順位）:
  1. RecordedAmount_Sum__c（累計計上金額）
  2. FundManagement__c の RecordedAmount__c 合計
  3. BillingAmount_Sum__c（累計請求金額）
  4. FundManagement__c の BillingAmount__c 合計
```

