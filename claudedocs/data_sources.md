# データソース情報

## スクレイピング対象

### 固定ソース
- **ハローワーク** - 常にスクレイピング対象として固定

### 変動ソース
- その他の媒体はタイミングによってまちまち
- 案件ごとに対象サイトが変わる

## データフロー

```
スクレイピング（ユーザー担当）
        ↓
    CSVファイル
        ↓
このシステムで処理
        ↓
  Salesforce同期
```

## 備考
- スクレイピング自体はユーザーが担当
- このプロジェクトはCSVファイルを受け取るところから処理開始
- 突合キーは電話番号がメイン（会社名・住所は補助）
