### 3. 因果関係

```
【主因：提案総額の値上げ × サブスクモデル】

提案総額2倍以上の値上げ（介護+104%、医療+137%）
        ×
サブスク型（先行投資・返金保証なし）
        ↓
顧客の投資リスクが大幅増加
        ↓
投資懸念 +13.1pt、成果懸念 +3.2pt
        ↓
決裁者が投資判断できない（代表者商談 +10.1pt）
        ↓
懸念系失注が増加（+7.0pt）


【副因：プロダクト問題】

・2年間プロダクトの品質が変わっていない
・差別化に失敗している
        ↓
「サービスに価値を感じていない」+1.2pt


【副因：営業スキル】

・懸念払拭力の不足
・特定人員で大幅悪化（服部+45.3pt、嶋谷+33.5pt、北尾+16.2pt）
        ↓
成約率の個人差（TOP 21.5% vs WORST 2.1%）
```

## 1. ビジネスモデルの背景

### サービス特性

```
【ビジネスモデル】
・サブスク型採用コンサルサービス
・月額課金（先行投資型）
・返金保証なし
・成果保証なし

【競合との違い】
・人材紹介: 採用できたら払う → 顧客リスクなし
・当社サブスク: 先に払う＋保証なし → 顧客リスクあり

【プロダクトの課題】
・2年間プロダクトの品質が変わっていない
・差別化に失敗している
```

## 8. 問題の本質

### 構造図

```
【表面的な問題】
・懸念系失注率: 14.0% → 21.0%（+7.0pt）
・投資懸念: +13.1pt
・代表者商談: +10.1pt

【根本原因】
┌─────────────────────────────────────────┐
│  提案総額の値上げ（2倍以上）              │
│         ×                               │
│  サブスクモデル（先行投資・保証なし）      │
│         ↓                               │
│  顧客の投資リスクが大幅増加               │
│         ↓                               │
│  決裁者が投資判断できない                 │
│         ↓                               │
│  懸念系失注が増加                        │
└─────────────────────────────────────────┘

【プロダクトの課題】
・2年間プロダクトの品質が変わっていない
・差別化に失敗している
・この金額帯にしては成果保証・返金保証系の安心あるいは見える何かがない
・営業の現場感としてミイダスなどの心理テストを始めとしたサービスは「目に見える納品物」があり成約しやすいと仮説があったが
　これはこの辺にも絡んでいるはず

【営業の課題】
・提案の質向上が必要（感覚的訴求→ROI訴求型営業）
・懸念払拭力の不足（特定人員での大幅悪化）

【構造的ジレンマ】
・高単価だからこそROI・投資対効果の訴求が必要
・しかし成果保証がないから効果を明言できない
・明言するとプロダクト側を追い込む
・明言しないと売れない
→ 営業力強化起点だけでは解決できない構造的問題
```

## 9. 戦略オプション

### 構造的ジレンマの認識

```
【現状の矛盾】
・高単価になった → ROI・投資対効果を論理的に訴求する必要がある
・しかし成果保証がない → 「この効果が見込めます」と明言できない
・明言するとプロダクト側を追い込む（約束したのに達成できないリスク）
・明言しないと売れない

→ 営業力強化だけでは根本解決にならない
→ ビジネスモデル自体の見直しが必要
→ プロダクト・セールスの戦闘力の向上が必要
```

### 一般的な高単価SaaS/コンサルの対処法

| 手法 | 内容 | 保証との関係 |
|------|------|-------------|
| **事例訴求** | 「類似企業でX%改善しました」 | 保証ではなく実績として提示 |
| **シミュレーション** | 「御社の場合、想定効果は○○」 | あくまで「想定」として提示 |
| **PoC/トライアル** | 小規模で効果検証してから本契約 | 顧客がリスク判断できる |
| **成果連動型** | 基本料金＋成果報酬の併用 | 一部リスクを共有 |
| **途中解約条項** | 効果なければ○ヶ月で解約可 | 出口を用意 |

### やれること一覧

| # | 方向性 | 内容 | メリット | デメリット |
|---|--------|------|---------|-----------|
| A | **提案総額の見直し** | 介護業界向け価格の引き下げ | 投資懸念減少、成約率向上 | 収益低下 |
| B | **プロダクト/ビジネスモデルの変更** | 保証制度、トライアル、成果連動型導入 | 顧客リスク低減、構造的解決 | 開発コスト、収益構造変化 |
| C | **営業力の強化** | 事例訴求、提案の質向上 | 現行モデル維持 | 効果限定的（構造問題は残る） |

### 詳細

#### A. 提案総額の見直し

| 施設タイプ | 現状 | 検討方向 |
|-----------|------|---------|
| 介護（高齢者） | 118.0万円 | **引き下げ検討**（業界の資金事情） |
| 医療 | 160.7万円 | 据え置きまたは引き上げ可能 |

#### B. プロダクト/ビジネスモデルの変更

**顧客の投資リスクを下げる施策：**

| 施策 | 内容 | 効果 |
|------|------|------|
| トライアル期間 | 1ヶ月無料または低価格で試用 | 効果を確認してから本契約 |
| 成果連動型料金 | 基本料金＋採用成功報酬 | リスク共有、顧客の納得感 |
| 返金保証 | 効果なければ○ヶ月以内返金 | 投資懸念の直接解消 |
| 途中解約条項 | 3ヶ月ごとに解約可能 | 長期コミットのハードル低下 |

#### C. 営業力の強化（効果は限定的）

**現行モデルでできること：**

- 事例の武器化（「保証はしないが、実績はこう」）
- 深堀（成約率21.5%）のノウハウ展開
- 投資懸念払拭スキルのトレーニング
- ROIシミュレーション資料の整備（「想定」として提示）

**限界：**
- 構造的ジレンマは解消されない
- 「明言できない」制約の中での改善に留まる

## 10. 経営判断事項

### 最重要：ビジネスモデルのジレンマ解消

```
【現状】
高単価 × 成果保証なし × ROI訴求必要 = 構造的に売りにくい

【判断が必要な問い】
・営業力強化だけで乗り切るのか？
・ビジネスモデル自体を変えるのか？
```

### 判断事項一覧

| 項目 | 検討内容 | 判断基準 |
|------|---------|---------|
| **ビジネスモデル** | トライアル/成果連動型/返金保証の導入 | 構造的ジレンマの解消 vs 収益構造変化 |
| **提案総額** | 介護業界向け価格の見直し | 業界の資金事情、競合価格 |
| **プロダクト** | 2年間変化なしの解消、差別化 | 開発リソース、優先度 |
| **営業施策** | 事例訴求強化、提案の質向上 | 構造問題が残る中での現実解 |

## 12. 注意事項・分析の限界

### データの制約

| 項目 | 内容 |
|------|------|

| 提案金額 | 単価×期間の内訳は不明 |
| 期間データ | 契約期間の変化は未分析 |

### 解釈上の注意

- 成約率（9.5% → 9.9%）自体はほぼ横ばい
- 懸念系失注「率」の増加 ≠ 懸念系失注「数」の増加とは限らない
- 提案総額値上げの収益影響は未分析
